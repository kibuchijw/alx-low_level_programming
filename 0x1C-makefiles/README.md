## Tasks
### 0. make -f 0-Makefile
* Create the first Makefile.
* Requirements:
	* name of the executable: `school`
	* rules: `all`
		* The `all` rule builds the executable
	* variables: none
### 1. make -f 1-Makefile
* Requirements:
	* name of the executable: `school`
	* rules: `all`
		* The `all` rule builds the executable
	* variables: `CC`,`SRC`
		* `CC`: the compiler to be used
		* `SRC`: the .c files
